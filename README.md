JFXtras-labs
============
_A project for experimenting with new controls and other useful JavaFX extensions_

Building with Gradle
--------------------
* Set your environment variable JAVAFX_HOME to the directory that contains rt/lib/jfxrt.jar.
* cd jfxtras-labs
* gradlew build
